# Twitch Viewer Bot

Бот подключается к IRC-каналу Twitch от имени вашего аккаунта, имитируя просмотр трансляции.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

При запуске бот попросит ввести:
1. **OAuth токен** — токен вашего аккаунта (формат: `oauth:abc123...`)
2. **Канал** — название канала, который нужно "смотреть"

## Получение OAuth токена

1. Перейдите на https://twitchapps.com/tmi/
2. Авторизуйтесь через Twitch
3. Скопируйте полученный токен (начинается с `oauth:`)

## Как это работает

- Бот подключается к IRC-серверу Twitch с вашим токеном
- Заходит в чат указанного канала
- Twitch считает это как активного зрителя
- Соединение поддерживается автоматически (пинг каждые 4 минуты)
- При разрыве соединения — автоматическое переподключение

## Несколько аккаунтов

Для просмотра с нескольких аккаунтов запустите несколько копий бота с разными токенами.

## Остановка

Нажмите `Ctrl+C` в консоли.
